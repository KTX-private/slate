# Errors

## 错误码

* -1xxx 网络或服务器故障
* -2xxx 请求, 头或参数错误
* -3xxx 请求无法被正确处理

| Code | 描述 |
| :-----: | :------: |
| -1000 | Unknown |
| -2001 | 未给定请求头 |
| -2002 | 请求头的值无效 |
| -2003 | 未给定参数 |
| -2004 | 参数的值无效 |
| -2005 | Body 含有无效的 Json 格式 |
| -2006 | 无效的签名 |
| -2101 | 已超出规定的 API 访问次数限制 |
| -2102 | 已超出规定的 API 用量限制 |
| -2103 | ApiKey 未被授予相关权限 |
| -2104 | 请求过期 |
| -3001 | 指定的资产不存在 |
| -3002 | 指定的交易对不存在 |
| -3003 | 指定的委托不存在 |
